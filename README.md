# Buyhatke Price Tracker

A Python script to fetch live data from Buyhatke.com and extract trending products (e.g., iPhone 15/16 prices from Flipkart/Amazon).

## Quick Start
1. Install: `pip install -r requirements.txt`
2. Run: `python parse_html.py`
3. Output: `output.json` (data) and `products.csv` (Excel-ready).

## Example Products
- iPhone 16 (Black, 128 GB): ₹57,999 on Flipkart
- iPhone 15 (128 GB): ₹47,999 on Amazon

Built with requests, BeautifulSoup. For personal use.
